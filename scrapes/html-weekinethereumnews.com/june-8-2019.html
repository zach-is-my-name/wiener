June 8, 2019

Ethereum News and Links

Layer 1

  • Prysmatic Eth2 client update – updating to v6 spec, working on Eth2 data API
  • Artemis Eth2 client update – up to date on v5 spec
  • Matt Slipper’s Eth2 test runner update
  • A plan to natively integrate Eth1 into Eth2
  • Justin Drake’s idea for simpler hash-efficient sparse tree
  • Raul Jordan’s Eth2 history and high level overview
  • Alex Beregszaszi’s history of Ewasm

Layer 2

  • 550 transactions per second using STARKs at only ~6000 gas per trade. Starkware and 0x release StarkDEX demo running on Ropsten. To be released on mainnet “in several months.” Code.
  • Plasma EVM with Continuous Rebase

Stuff for developers

  • Verisol: Microsoft open sources Solidity formal verification tool using its IR Boogie language
  • genSTARK: a JavaScript zk-STARK generation framework
  • Remix IDE v0.8 – new layout
  • An open node for Augur markets
  • Building upgradeable smart contracts
  • Truffle v5.0.21
  • OpenZeppelin v2.3
  • Getting the most out of CREATE2

Ecosystem

  • Quickblocks/Trueblocks v0.5.7. full blockchain scraper to build fully decentralized block explorer, ie get trustless data off your own node
  • Tripathi: Eth tokens as Wall Street backoffice replacement

Enterprise

  • Blockchains to unlock $3.1 trillion in business value by 2030, per Gartner.
  • Paul Brody talk: what blockchain can deliver now
  • Epirus block explorer for Quorum, Ethereum and Azure
  • Quick start a private chain with Quorum and Truffle
  • Fnality, formerly Utility Settlement Coin, raises 50m GPB from its consortium of banks.
  • “Carrefour says blockchain tracking boosting sales of some products,” per Reuters
  • A guide to start using Pantheon, the enterprise focused Eth Java client

Governance and Standards

  • EIP2098: Compact 64 byte representation of Eth sig
  • ERC2100: Streaming token standard
  • ERC2095: Temporary Data Persistence

Application layer

  • Augur’s 4 prong strategy to defeat invalid market scams
  • Numerai’s Erasure Quant
  • OpenLaw: creating and transferring tokenized art donations
  • Request Protocol v2 beta release
  • Grid+ passes 1000 in power customers
  • FunFair is doing an affiliate program
  • Dai Stability Fee lowered to 16.5%

Interviews, Podcasts, Videos, Talks 

  • Videos from Scaling Ethereum
  • Latest decentralized data community call with demos from Quickblocks and Play
  • Latest curation markets call
  • Scuttlebutt’s Dominic Tarr on Epicenter
  • Kyber Network’s Loi Luu on Into the Ether

Tokens / Business / Regulation

  • SEC charges Kik with for unregistered security sale. Kik’s press release response. Coincenter says it’s a question of fact, not law.
  • Blockchains LLC acquires Slockit
  • Scenario analysis of SNT for ENS usernames
  • Cadence’s tokenized private notes get listed on Bloomberg terminal
  • cadCAD: Complex Adaptive Dynamics Computer Aided Design
  • Pairwise coordination subsidies. Vitalik’s quadratic funding twist to thwart collusion
  • de la Rouviere: Patronage as a game and an asset class

General

  • A walk down Ethereum and ConsenSys memory lane with Andrew Keys
  • Dether is adding chat to its in-person exchange app using IPFS, pinning service, and OrbitDB
  • LocalBitcoins ended in-person cash sales, so LocalEthereum does a no fee promo for in-person cash sales in June
  • What to do when sim swapping happens to you
  • According to his writing, Satoshi built Bitcoin for payments
  • Harz, et al, IACR paper: reduce deposits while maintaining security
  • Kim, Lee, Oh IACR: QAP-based Simulation-Extractable SNARK with a Single Verification
  • Gabizon IACR: Improved prover efficiency and SRS size in a Sonic-like system
