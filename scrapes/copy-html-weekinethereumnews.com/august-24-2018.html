August 24, 2018

News and Links

Protocol

  • [Shasper] Psymatic’s biweekly update
  • [VDF] some requirements for an Ethereum 2.0 VDF
  • [Casper] Latest Casper standup
  • [Plasma] OMG’s Plasma implementation in Elixir
  • [State channels] Ledger channels and virtual channels

Stuff for developers

  • Truffle v5.0.0 released. web3js 1.0, bring your own compiler, and lots more
  • Augesco, a drizzle competitor
  • Truffle University – graduates get a job as a blockchain dev or their money back. 120 applications in the first day
  • Engineering at Parity. and Q&A with CTO Fred Harrysson
  • Ocaml bindings to Geth
  • Civil’s event crawler in Go
  • Eth development with Go
  • EtherVM: EVM opcodes and disassembler
  • 2dproxy: delegatecall proxy that delegatecalls the constructor
  • Patrick McCorry’s battleship state channel code
  • NuCypher releases nuFHE (fully homomorphic encryption) library with orders of magnitude efficiency improvement
  • Gitcoin requests: ask for a bounty on any Github issue
  • Robbie Bent’s insights from security interviews
  • Airswap dev tutorial, part 3
  • Video: new features of Solidity v0.5
  • It’s now pip install vyper. And a video intro to Vyper
  • Frame’s “developer prerelease” on testnet to allow easier and lower level hardware wallet integration
  • Wireshark dissectors for Ethereum’s devp2p is now open source

Releases

  • Geth v1.8.14 with Swarm v0.3.2: A full miner rewrite (!) with instant insertion of uncles into work packages which makes safer the current larger blocks

Live on mainnet

  • Realit.io oracle is live – “Snopes meets Mechanical Turk on a blockchain”
  • Rock paper scissors on Ethereum

Ecosystem

  • It’s raining grants: Ethereum Foundation round 3. And ECF round 2. Plus Aragon round 2, as well as their update on rd1 grants.
  • uPort’s 3box – manage your public and private data using IPFS and OrbitDB
  • Very cool Alethio graphic: function calls on Ethereum over time
  • the web3 finance stack graphic
  • the web3 video stack
  • Dogethereum bridge alpha to livestream on Sept 5th

Governance and Standards

  • Today’s core dev call will feature discussion on mining reward reduction
  • Afri on ProgPoW
  • ERC948 architecture choices for subscriptions
  • ERC1341: Ethereum web tokens
  • EIP1150: secure delegatecall

Project Updates

  • Multi-collateral Dai goes live on Kovan on September 17th
  • Golem: verifying rendering results
  • Peepeth’s kickstarter is live (or use Ether). Shooter Jennings mentioned Peepeth on Joe Rogan
  • OmiseGo Aug update
  • OpenLaw using Chainlink as the oracles for their legal contracts
  • iExec has an interesting weekly series on dapps built on their platform
  • Aragon Q2 transparency report
  • Ujo Music: exploring Radical Markets, patronage & collecting
  • Livepeer’s Tributary milestone release

Interviews, Podcasts, Videos, Talks

  • Kevin Owocki on Hashing It Out
  • Ryan Zurrer on Epicenter
  • Ryan Berckmans talks his Predictions Global frontend for Augur
  • Matthew Di Ferrante on Zero Knowledge
  • Open web3 design call video
  • A readable version of Brendan Eich’s AMA. hints at some interesting stuff.
  • Joseph Lubin media this week: Hidden Forces podcast, TDAmeritrade Network, and with Rhys Lindmark
  • Meher Roy with Jake Brukhman
  • More TechCrunch Eth Zug vids: Vlad Zamfir on governance, Karl Floersch on cryptoeconomics, and exchange/dex panel

Tokens / Business

  • Sam Cassatt: misconceptions in the Ether(eum) market
  • Gnosis: Ethereum is blooming and we are doubling down on it
  • Someone won 10000 ETH from FoMo3d by reverse engineering the bots so they didn’t respond when blocks got stuffed. tl;dr: large transactions at low-ish gas prices to fill most the blocks but then super high gas price smaller transactions to fill the block to 100%.
  • Contrary to BTC maxi narratives, there’s no Eth “ICO firesale”
  • Panvala: token experiments at the ConsenSys retreat
  • TCRs: value prop and challenges

General

  • A bunch of Bitcoin ETFs got rejected by SEC. Commissioner Peirce says those were staff actions that will get reviewed
  • web3Foundation’s call for grants
  • 19 year old arrested for sim swapping to steal crypto
