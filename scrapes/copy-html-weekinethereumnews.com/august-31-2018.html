August 31, 2018

News and Links

Protocol

  • Last week’s core dev call with substantial issuance reduction conversation from the community. Notes. [As I press send, a decision was made that a reduction to 2 ETH was the clear community consensus]
  • [Eth 2.0] Latest Eth 2.0 implementer call. Agenda.
  • [Eth 2.0] Ethereum 2.0 dev onboarding
  • [Eth 2.0] Scalability so far in 2018 by Ben Edgington
  • [Sharding] Simple synchronous cross-shard transaction protocol
  • Vitalik: “layer 1 should be innovative in the short term but less in the long term”
  • [Plasma] OmiseGo’s comprehensive Plasma update #2
  • [Plasma] Latest Plasma implementer call
  • [Plasma] Loom Network’s architecture: Plasma Cash chain with two subchains
  • [Plasma] Plasma MVP in Vyper
  • [Plasma] A Plasma Debit explainer
  • [State channels] State channels for babies
  • [Moon] Victor Maia’s first post-Devcon3 update on Moon. Massively parallel, formally verified, a VM running on GPU with a native browser in Rust? Wow

Stuff for developers

  • LibSubmarine: Temporarily hide transactions using GasToken
  • Etherlime v.0.9 dev framework using ethers.js, now supports unit tests with code coverage. Etherlime tutorial
  • Revamped Embark documentation
  • Getting started with Drizzle + React tutorial
  • Solidity gas optimization tips
  • Ethereum in Google’s BigQuery, the ethereum_blockchain dataset updates daily. The dataset is also available at Kaggle.
  • Mythril to become a platform API for professional users
  • Loredana Cirstea meetup presentation of Pipeline visual IDE (video)
  • Origin messaging service using OrbitDB (they pin IPFS messages pre-Filecoin). Sign a message to create an Origin message private key that lives in local storage

Releases

  • Geth v1.8.15: miner fixes and more aggressive uncle inclusion. There’s also now a Geth discord channel
  • web3.js 1.0.0-beta.36

Enterprise

  • Quorum v2.1.0
  • Blk.io’s Epirus enterprise platform – Quorum, Crux, an enterprise focused block explorer
  • EEA hackathon in Prague after Devcon

Live on mainnet

  • Town Square– a web3 version of Disqus using Eth + IPFS

Ecosystem

  • New Reddit tipbot supports Dai, KNC, OMG, REQ and DGX
  • Wyre is going to start issuing a kyc/aml NFT for dexes to use
  • Emre Tekisalp: understanding web3, a user-controlled internet

Governance and Standards

  • Core dev call on Constantinople. Decisions made: issuance reduction to 2 ETH per block, 12 month difficulty bomb delay, ProgPoW to be studied for next hard fork
  • Initial code for ERC1337’s subscription model
  • Aragon Labs governance ideas and experiments
  • EIP1355: Ethash1a minimal changes to thwart asic mining
  • ERC1361: Standardized AppLink for signing ETH Transactions on mobile
  • ERC1363: Payable token for ERC20, a la ERC721

Project Updates

  • With multi-collateral Dai, you can get interest for holding Dai
  • Raiden v0.6.0 webUI redesign and bug fixes
  • UX challenges in Golem and beyond
  • Storj launches v3 private alpha
  • Civil to license AP content for its newsrooms
  • Brave passes 10 million downloads just on Android. That’s partly because private mode on other browsers isn’t private at all. If you haven’t switched to Brave yet, here’s my referral code.
  • Aragon: your own personal DAO

Interviews, Podcasts, Videos, Talks

  • Jacob Czepluch Raiden presentation at Copenhagen meetup
  • Yahoo interview with Joseph Lubin (7m)
  • Gavin Wood Bloomberg interview (5m)
  • Maker DAO’s Rune Christensen on Stable Report
  • Benedikt Bünz talks VDFs and bulletproofs on ZeroKnowledge
  • Augur’s Joey Krug with Laura Shin
  • Joseph Lubin and Jimmy Song discuss the terms of their bet

Tokens / Business

  • The Melon token design
  • Ambrož Homar: Is changing token models breaking promises or innovation?
  • Meridio: smart contracts and multi-party leases
  • A multi-phase bonding curve token model for decentralised SaaS
  • Pauk Kohlhaas: fund, curate and distribute IP with NFTs and curation markets

General

  • CBOE is telling market makers that Ether futures are coming soon
  • Reuters reports on the many layers of scam that is Venezuela’s Petro
  • Inside Coinbase’s key generation ceremonies
  • Elon Musk tweets: “At this point, I want ETH even if it is a scam”
  • Tim Swanson quantifies the electricity used by proof of work blockchains
  • The rise of crypto in higher education
